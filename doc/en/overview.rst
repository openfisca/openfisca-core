****************
Package overview
****************

:mod:`openfisca` consists of the following things

 * A library to microsimulate the taxes and transfers of any kind of fiscal system
 * Taxes and transfers of any kind of test case household can be simulated
 * If survey microdata is provided, the microsimulation can deal with an entire population
 * A graphical interface to interact with the library
 * Legislative reforms can be explored for test case or the whole population when survey data is provided
 * A large part of France socio-fiscal system is available since 2002
 * There is some ongoing work on a Tunisian system

may 2011 with some support from `IDEP <http://www.amse-aixmarseille.fr/en/presentation/idep>`_  . It was open-sourced at the end of 2011.
The Centre d'analyse stratégique and IDEP continue to provide resources for development.

OpenFisca is only made possible by a group of people like you
who have contributed new code, bug reports, fixes, comments and ideas. A
complete list can be found `on Github <http://www.github.com/openfisca/openfisca/contributors>`__.
